Mac Installation

Hopefully this won't take too long.

Check the Dependencies

Most Macs come with a lot of the stuff we need pre-installed. Let's make sure yours does, too.

Open a command line and type the following commands. If a mesage like "command not found" gets returned, that means you'll need to install it.

ruby -v

If ruby isn't installed, you can get it here: https://www.ruby-lang.org/en/documentation/installation/#homebrew.

Once you've verified that ruby is installed, install a few gems we'll be using:

gem install doc_raptor

Tip: If any of these steps ever fail and you get a "Permission denied" warning, try adding the word "sudo" before the command, like this: sudo gem install doc_raptor.

python -V

If python isn't installed, you can get it here: https://www.python.org/downloads/release/python-2710/.

java -version

If java isn't installed, you can get it here: https://www.java.com/en/download/help/mac_install.xml

xsltproc -version

If xsltproc isn't installed, you can get it here: http://www.sagehill.net/docbookxsl/InstallingAProcessor.html

Now install Prince. Download the zip file: http://www.princexml.com/download/prince-10r2-macosx.tar.gz, and run the "install.sh" script that is included.

Run the Installer

Next, make a folder where you want to run bookmaker, download these three zip files and put them inside it:

https://github.com/oreillymedia/HTMLBook/archive/master.zip

http://sourceforge.net/projects/saxon/files/latest/download?source=files

[our zip file]

Double click to unzip all three zip files, but don't do anything else to them.

Now let's run the installer to get everything up and running. We're going to use some preset folder and config settings, just for the sake of time, but everything is configurable--you can see the full docs at http://github.com/macmillanpublishers.com/bookmaker. TIP: You can just drag the install.rb file onto your Terminal window instead of typing the full path.

ruby /path/to/install.rb

That's it, my friend. It is done. Use the new "convert" folder to put your files for conversion, and the finished files will show up in the "done" folder.

Oh, except if you want to enable epub image processing, you should also install image magick: http://cactuslab.com/imagemagick/. But this isn't required and we have it turned off by default.
